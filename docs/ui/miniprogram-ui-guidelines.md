# 微信小程序 UI 统一方案

## 视觉方向

- 关键词：简洁、安静、专业、轻作品集
- 目标：让内容本身成为主角，页面承担秩序和气质，不抢戏
- 风格：暖白底、深墨文本、森林绿点缀、低饱和辅助橙

## 颜色

- 页面背景：`#F6F4EE`
- 卡片背景：`#FFFCF7`
- 浮层背景：`rgba(255, 252, 247, 0.92)`
- 主文字：`#1F2320`
- 次文字：`#66706A`
- 弱辅助文字：`#8A918C`
- 主色：`#2F6B57`
- 主色深态：`#245443`
- 边框：`#DDD8CC`
- 点缀色：`#C96F4A`

## 配色策略

- 大面积背景始终保持暖白，避免纯黑或深蓝大面积压住内容
- 绿色只用于主按钮、选中态、关键标签，不铺满整页
- 橙色只用于危险操作或少量提醒，不参与常规导航
- 文字层级只保留三档：主文字、次文字、弱辅助文字

## 排版

- 页面大标题：`42rpx` / `700`
- 区块标题：`34rpx` / `700`
- 卡片标题：`28rpx` / `700`
- 正文：`24rpx` 到 `28rpx`
- 标签与时间：`22rpx` 到 `24rpx`
- 字体：`"Avenir Next", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif`

## 组件

- 卡片圆角：`24rpx`
- 英雄区圆角：`32rpx`
- 胶囊按钮：`999rpx`
- 图标容器：`76rpx * 76rpx`
- 阴影：`0 8rpx 24rpx rgba(31, 35, 32, 0.08)`

## 按钮

- 主按钮 `btn-primary`：深绿渐变底白字，用于核心跳转和核心操作
- 次按钮 `btn-secondary`：浅底描边，用于辅助跳转与次级提交
- 第三级按钮 `btn-tertiary`：中性浅底，用于不强调的辅助操作
- 危险按钮 `btn-danger`：低饱和橙红，用于删除和不可逆操作
- 筛选按钮 `btn-chip`：用于分类、标签、切换选项，选中态叠加 `btn-chip-active`
- 文本按钮 `btn-text`：只用于分区右侧的轻跳转，例如“全部内容”

## 按钮尺寸

- 标准按钮高：`76rpx`
- 紧凑按钮高：`64rpx`
- 标准圆角：`999rpx`
- 内边距：横向 `24rpx` 到 `28rpx`

## Icon 规范

- 风格：线性或极简字符图标，不混合面性图标
- 视觉重量：细线条、中等留白、统一圆角容器
- 推荐语义：
  - 首页：`home`
  - 技能：`spark`
  - 作品集：`gallery`
  - 关于我：`profile`
  - Blog：`note`
  - Vlog：`play`
  - 联系方式：`message`
  - 管理入口：`shield`

## 底部四分区按钮

- 结构：悬浮式底部导航，而不是默认平铺 tab
- 外层：暖白半透明背景、圆角 `36rpx`、轻毛玻璃、柔和阴影
- 单个按钮：上图标下文字，激活时增加浅绿选中底
- 激活图标容器：绿色渐变底，未激活保持浅底卡片感
- 文案：`首页 / 技能 / 作品集 / 关于我`

## 页面策略

- 首页：个人信息、两个核心动作、精选媒体、最近更新
- 技能页：按分类展示技能卡片，用条形熟练度代替星级
- 作品集：顶部简介 + 胶囊筛选 + 双列媒体网格
- 关于我：介绍卡片 + 分段切换 + 内容列表 + 低可见度管理入口
